GALAGA

This game is produced during "I'm a Developer!" Event in GDG Suwon.
This event is the parady of "I'm a singer!" on MBC.

Key Buttons

Arrow Key : Move
"space" : Bullet
"z" : Bomb
"x" : Shutgun
"c" : Shild
"shift" + "arrow" : Homming a.k.a 'torpedo number 1' of North Korea.
